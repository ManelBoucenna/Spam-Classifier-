# Spam-Classifier

Classifying the famous Spam / Ham SMS Data from UCI Machine Learning Dataset with SKLearn.

Data : https://archive.ics.uci.edu/ml/datasets/sms+spam+collection

* Used SKLearn
* Spot Checking Algos
* Heatmap for Naive Bayes
* Acheived 98.8% Accuracy
